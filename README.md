## 介绍

基于 [wangEditor](https://www.wangeditor.com/) 封装的开箱即用的 [Vue2 组件](https://www.wangeditor.com/v5/for-frame.html#vue2)

## 安装

```shell
yarn add @wangchaodeyuzhou/editor
yarn add @wangchaodeyuzhou/editor-for-vue
```
